# @opengui/backend

Networked **OpenGUI Backend**: embeds `@opengui/runtime`, adds HTTP/WebSocket/SSE (and Desktop IPC), queued prompts, arbitration, Backend persistence.

**Status:** Scaffold package only. The running Backend today is `server/web-server.ts` (see [`docs/architecture.md`](../../docs/architecture.md)). Physical move into this package: [`docs/plans/runtime-backend-sdk-split.md`](../../docs/plans/runtime-backend-sdk-split.md) Phase 4.
